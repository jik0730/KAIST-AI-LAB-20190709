# Before going on Tutorial...

1. 현재 git repository를 clone하고 해당 폴더로 이동합니다.

* git clone https://github.com/jik0730/KAIST-AI-LAB-20190709.git

2. 필요한 python package를 설치합니다.

* pip install jupyter
* pip install matplotlib
* pip install pillow
* pip install numpy
* pip install scipy
* pip install sklearn

3. Jupyter notebook을 실행합니다.

* jupyter notebook

4. 실습 파일로 이동합니다.
